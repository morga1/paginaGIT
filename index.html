Practica de GIT

git pushManda los archivos del area de repositorio a un servidor, por ejemplo github, bitbucket, gitlab, etc.
Comando Descripcion
git init Inicializa la carpeta como un repositorio de GIT, para empezar a gestionar el cambio.
git log Permite visualizar todos los commit realizados en un repositorio.
git add -A Agrega todos los archivos que han sido modificados, o archivos nuevos del working area al stagging area.
git commit -m "" Pasa los archivos del area de stagging al area de repositorio, es necesario poner un mensaje que represente todos los cambios del proyecto.
